# wqs-slider

#### 项目介绍
🎉一个微信小程序滑动效果，仿探探卡片动画效果分享

![gif](http://qiniu.wqsong.com/uniapp-example.gif)

#### 安装说明

1.  git clone
2.  将project.config.json中的appid修改为您的appid
3.  使用微信开发者工具打开根目录即可查看效果
